# Ink & Steel: The Librarian's Tale 📚⚔️

This repository contains the full design framework for a turn-based JRPG, developed as a technical exercise for Game Design.

## 📑 Project Overview
"Ink & Steel" is a fantasy-comedy JRPG focused on a dual-character synergy mechanic. The project includes full documentation: **GDD, LDD, and TDD.**

## ⚙️ Key Design Pillars
* Defined movement metrics (3 tiles/s walking, 5 tiles/s running) and interaction heights.
* Detailed stat-scaling for the "Synergy System" and basic turn-based combat damage formulas.
* Designed learning curves and level flow, transitioning from safe zone to high-intensity combat area for the first level.
* Implementation of State Machines for character states (Idle, Walk, Run, Combat) and pseudocode for core systems.

The complete documentation, including hand-drawn level layout and technical pseudocode, is available in the PDF file above.
