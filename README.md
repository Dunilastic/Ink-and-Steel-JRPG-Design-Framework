# Ink & Steel: The Librarian's Tale

This repository contains the full design framework for a turn-based JRPG, developed as a technical exercise for Game Design. The complete documentation (GDD, LDD, and TDD) has been fully translated into English to showcase international project standards.

## Project Overview
"Ink & Steel" is a fantasy-comedy JRPG focused on a dual-character synergy mechanic. The project includes full documentation: **GDD, LDD, and TDD.**

## Key Technical & Design Pillars
* System Architecture: Detailed pseudocode for combat engines, turn-based logic, and dynamic audio triggers.
* Finite State Machines (FSM): Structured logic for character states (Idle, Walk, Run, Combat) and Enemy AI behavior.
* Level Design Metrics: Precise grid-based movement and interaction heights.
* Balancing & Scaling: Simple mathematical formulas for the "Synergy System" and combat damage calculations.
* Pacing & Progression: Defined learning curves, transitioning from safe zones to high-intensity combat areas of the first "level".

## Documentation
The complete PDF includes:
* GDD: Core loops, mechanics, and meta-narrative.
* LDD: Hand-drawn level layout and asset lists.
* TDD: Script architecture and hardware requirements.
